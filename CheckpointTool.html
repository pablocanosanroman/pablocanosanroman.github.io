Unreal Engine FPS puzzle

In this page, I show the checkpoint tool that I created for games mechanics programming using live code(C++ and Blueprints) in Unreal Engine 5. The blueprints have only been used to create the utility widget to control the tool. This tool allows you to create checkpoints wherever you want on the world, replace other gameObjects with checkpoints, control if the checkpoints are interactable or not (if they need input from the player to activate them), control if the checkpoints will activate a particle system when they get activated or not, and control if the checkpoints have an order of activation or not. In case that the order of activation is on, the user will be able to visually see the order of activation when the checkpoints are added to the checkpoints array.
